Agents in parallel

A tiled canvas of live terminals. Run several coding agents at once and see who's working, idle, or waiting on you at a glance.

Git-worktree isolation

Every agent gets its own branch + worktree, so parallel edits never clobber each other.

One-click diff & merge

Review each agent's changes in-app and merge into your base branch — or discard — without leaving the canvas.

Make it yours

Liquid-glass UI with your own accent colour, wallpaper, and adjustable terminal transparency.
