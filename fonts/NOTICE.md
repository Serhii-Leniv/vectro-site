# Bundled fonts

These typefaces power the Vectro type system (see `@font-face` in `styles.css`).
Both are redistributed under the **SIL Open Font License 1.1** (OFL), which permits
bundling in an application as long as this attribution travels with them.

| App family       | Upstream typeface | Author / Copyright                          | License |
| ---------------- | ----------------- | ------------------------------------------- | ------- |
| `Vectro Sans`    | Inter (variable)  | Copyright The Inter Project Authors         | OFL 1.1 |
| `Vectro Display` | Space Grotesk     | Copyright The Space Grotesk Project Authors | OFL 1.1 |

Files are the `latin` subset, variable-weight `woff2` builds pulled from the
Fontsource CDN. Full license text: https://openfontlicense.org

Inter: https://github.com/rsms/inter ·
Space Grotesk: https://github.com/floriankarsten/space-grotesk
